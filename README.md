# Fredy Furtado Salon App

Sistema de gestão para salão de beleza desenvolvido em Python com Streamlit.

## Funcionalidades
- Cadastro de Clientes
- Agenda de Atendimentos
- Controle Financeiro (Pix, Cartão, Dinheiro)
- Relatórios Financeiros (Diário, Semanal, Mensal, Anual)
- Cadastro de Equipe e Serviços
